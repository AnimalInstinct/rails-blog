# Blogs application
## Versions

- ruby 3.0.0
- rails 6.1.3.2
- bootstrap 5

## Features

- [Deployed at Heroku](https://bmdonego.herokuapp.com/).
- Sqlite for the development environment and Postgresql for the production.
- Custom authentication and authorization system with only bcrypt depends on.
